#####任意目录下打开DOS窗口

  • 按住Shift右键—-在此处打开命令窗口
    #####常见DOS命令
  • 盘符切换—-d: 回车
  • 列出当前目录文件或目录—-dir
  • 创建目录—-md 目录名
  • 删除目录—-rd 目录名
  • 删除不为空的目录—-rd /s 目录名 或 rd /s /q 目录名
  • 返回到根目录—-cd\ 回车
  • 删除文件—-del 文件名
  • 清屏—-cls 回车
  • 退出—-exit

#####JDK、JRE、JVM关系

  • JDK （Java Development Kit Java开发工具包）
  • JRE （Java Runtime Environment Java运行环境）
  • JVM （Java Virtual Machine Java虚拟机）
  • JDK包括JRE，JRE包括JVM
